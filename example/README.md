# README
This folder contains some examples, without the extension the code blocks below and files in this folder should all contain no highlighting, after you install the extension there should be highlighting.


## Code Block Example

block with `gr`:
```gr
module Main
let test = () => print("test")
test()
```

block with `grain`:
```grain
module Test
enum Direction {
  North,
  South,
  East,
  West
}
```
